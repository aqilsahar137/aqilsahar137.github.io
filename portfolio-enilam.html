e-Nilam

Project information

  • Category: Online Book Recording System
  • Purpose: Group Project
  • Project date: Dec. 2021
  • Skills involved: JSP (JAVA SERVER PAGES), HTML, SQL, JAVA (PROGRAMMING LANGUAGE)

About the project:


e-Nilam (short for Electronic Nilam) is an online book recording system that acts as a step forward in the already-existing Nilam effort that is currently being used in schools. Through the system, students can record their reading history and the records will be kept online. This eliminates any form of worry regarding the keeping of a physical record that can potentially be lost due to recklessness.

Group Members involved in the project: AMALIA NATASHA BINTI MOHAMAD PUAT, MEGAT NOOR AIDIL PUTRA BIN NOOR AZMAN, MUHAMMAD AQIL BIN MOHD SAHAR, NUR ATIQAH BINTI SHAMUDDIN, and ZAKIRUL HAFIZ BIN KHAIRUDIN SHAH.


Purpose of the project:


This project was assigned to us for exploration purposes on how to set up a proper system using Java Server Pages. This online book recording project was completed using JSP (JAVA SERVER PAGES), HTML, SQL, and JAVA (PROGRAMMING LANGUAGE).


Challenges faced during development:


The obstacles came in the form of integrating HTML with JAVA programming language as JSP does not allow for the use of PHP and JAVASCRIPT. Working on this project was a very foreign experience for all of us, needless to say.
