Writing2Go

Project information

  • Category: E-Commerce Site
  • Purpose: Degree Project
  • Project date: Jan. 2022
  • Skills involved: PHP, HTML, SQL, JQUERY, JAVASCRIPT, BOOTSTRAP

About the project:


Writing2Go is an online stationeries supply shop project that aims to carry a variety of essential stationery supplies. Featuring a variety of products, e.g. writing utensils, paper products, computer appliances and many more! Writing2Go is dedicated in providing the best quality stationeries at very affordable prices for buyers.


Purpose of the project:


This project was assigned as an assignment to have us explore the many ways to set up an E-Commerce site. The motivation of this website came from the hard times during lockdown when stationeries were not readily accessible in supermarkets and convenience stores around that time during the global pandemic. This mock E-Commerce site project was completed using BOOTSTRAP, HTML, CSS, PHP, JAVASCRIPT, JQUERY and MYSQL.


Challenges faced during development:


I faced a great deal of challenge in trying to figure out my own algorithm for Writing2Go's cart system, including how to add items into a cart and how to remove items from the cart. I probably spent the most of my time on figuring out the interconnection between all the database tables within the system and how they should behave. Such example would be when the admin decides to remove a user's account from the system. I had to make sure when that action is taken, it does not affect other tables that kept records such as payment history and order history.
