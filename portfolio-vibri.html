vibri - Self-Help Skills Tutor Application for Children with Autism

Project information

  • Category: Web Application
  • Purpose: Final Year Project
  • Project date: Feb. 2023
  • Skills involved: PHP, HTML, MYSQL, JQUERY, JAVASCRIPT, BOOTSTRAP

About the project:


vibri is a cross-platform web-based application developed to tutor self-help skills to children living with autism spectrum disorder. The application allows parents to gain access to professionally-backed video models that they can show to their children in order to teach them how to carry out various self-help skills (think showering, toileting...).


Purpose of the project:


The project serves as my entry for my Final Year Project to end off my bachelor's degree program in Computer Science Multimedia Computing at UiTM. The development of this project is aimed to make use of everything that I have learned thus far to produce a professional-looking and professional-feeling application in which it's main purpose is to deliver educational content to end users. For this project, I challenged myself to code a full-fledged web-based application using solely HTML, CSS, PHP, JAVASCRIPT, JQUERY and MYSQL.


Challenges faced during development:


My supervisor requested that I build this application from scratch without using trending front-end frameworks that are currently being used in the market as a test on whether I have fully mastered the basic programming languages for web development. However, I was allowed to use Bootstrap for presentation purposes. That, paired with my co-supervisor's expectations to have me produce a professional-looking website really pushed me to be creative with my approach on the project.

Trying to decide on the best visual approach for a tutor application is also a challenge in and of itself as I had to do some research on what passes for a good design.
